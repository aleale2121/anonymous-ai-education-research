# Anonymous Research Repository

This repository contains the topic modelling output and the code used to collect, preprocess, and analyze Reddit discussions about Artificial Intelligence in education.

The pipeline includes data collection, preprocessing, role classification, topic modeling using BERTopic, topic refinement, and thematic analysis.

# Outputs
The `outputs/` directory contains the results generated by the topic modeling pipeline. These results include the raw BERTopic topics, refined topic summaries, and aggregated analysis outputs used in the paper.

The most important files for interpreting the results are the **`refined_topics.csv`**
files, which contain the final topic summaries.

### **outputs/global**

Results from the global topic model trained on the full dataset.

Files include:
- **topics.csv**  
  Raw BERTopic output containing topic IDs, frequencies, and keyword representations.

- **refined_topics.csv**  
  Refined topic summaries generated during the topic refinement step. These summaries provide short descriptions of each topic and correspond to the interpretable topic labels used in the analysis.

- **dtm_topics_over_time.csv**  
  Dynamic topic modeling results showing how topics evolve across years.

- **topic_with_semantic_categories.csv**  
  Topics mapped to higher-level semantic categories.

- **topic_categories_over_time.csv**  
  Aggregated counts of topic categories over time used to generate the temporal
  analysis figures in the paper.

### **outputs/role**

Topic models trained separately for role-based subsets of the dataset.
```
outputs/role/
├── educator/
└── student/
```

Each role directory contains:

- **topics.csv** – raw BERTopic topic output
- **refined_topics.csv** – refined topic summaries

The **`refined_topics.csv` files in this directory correspond directly to the role-based topic tables presented in the paper**, where educator and student discussions are analyzed separately.

These files are the primary outputs used to interpret the role-based results.

### **outputs/yearly**

Topic models trained independently for each year (2018–2025).

Each yearly folder contains:

- **topics.csv** – raw BERTopic topics discovered for that year
- **refined_topics.csv** – refined topic summaries for that year

These models are used to analyze the temporal discourse patterns.

# Installation

Create a Python environment and install dependencies:

```
make install
```

or

```
pip install -r requirements.txt
```

---

# Pipeline Overview

The workflow consists of the following stages:

1. Data collection
2. Dataset merging
3. Data preprocessing
4. Role classification
5. Topic modeling
6. Topic refinement
7. Thematic categorization
8. Visualization

---

# Running the Pipeline

### Collect Reddit data

```
make collect
```

### Merge yearly datasets

```
make merge
```

### Preprocess dataset

```
make preprocess
```

### Role classification

```
make classify
```

---

# Topic Modeling

### Global topic model

```
make global
```

### Yearly topic models

```
make yearly
```

### Role-based topic models

Student topics:

```
make role_student
```

Educator topics:

```
make role_educator
```

---

# Topic Refinement

Refine topic labels using the LLM-assisted refinement pipeline.

```
make refine_global
make refine_student
make refine_educator
make refine_yearly
```

# Thematic Categorization

Group topics into higher-level semantic categories.

```
make categorize
```
